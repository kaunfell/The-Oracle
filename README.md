**The Oracle**
 https://kaunfell.github.io/The-Oracle/

This project was about learning to add some Javascript to HTML and to use Git Bash.
The Javascript code is generating random numbers which have different text values as answers by the Oracle.
When a user clicks "Ask" button, the question shows on the next line and the answer is triggered. The 
question area is cleared after pressing the "Ask" button so it easy to ask another question.
I used multiple pages for advices and ChatGPT.
